ABOUT US

DIMSUM.com adalah web yang memberikan informasi tentang alur distribusi secara tidak langsung dari awal sebagai admin serta sampai ke konsumen. Dengan adanya data informasi terkait alur distribusi sembako maka, apabila terdapat kejanggalan akan terlihat. Situs web ini mampu memberikan data grafik terkait lokasi peredaran sembako dan banyaknya jumlah sembako. Web ini juga akan mengelompokkan data distribusi berdasarkan jenis sembako. Dalam web ini terdapat opsi masuk dan daftar akun sebagai admin dan konsumen, kemudian alur distribusi sembako, dan pusat informasi dan bantuan.
